18. 배열을 다음과 같이 선언하였다. 행과 열의 수는 무엇인가?
``` c
int a[][4] = {{5,3}, {2,7}, {4,8}};
```
  - 3행 4열
    - 행의 크기가 생략되었으므로, 선언시 초기화 된 행의 개수인 3으로 선언되고, 열은 선언문에서 쓰인 것과 같이 4열이 된다.
